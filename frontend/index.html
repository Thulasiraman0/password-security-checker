    Password Security Best Practices

    DO:

    • Use at least 12 characters (longer is better)
    • Mix uppercase, lowercase, numbers, and symbols
    • Use a unique password for each account
    • Use a password manager to store passwords
    • Enable two-factor authentication (2FA)
    • Use passphrases: they're long and memorable
    • Change passwords if you suspect a breach

    DON'T:

    • Use personal information (names, birthdays, etc.)
    • Use common words or patterns (password123, qwerty)
    • Reuse passwords across multiple sites
    • Share passwords with others
    • Write passwords on paper or sticky notes
    • Use sequential characters (abc, 123)
    • Save passwords in browsers on shared computers

    Password Examples:

    Bad password123 john1990 qwerty
    Good Tr0pic@l-S7orm!2024 Purple$Elephant#Dance99 Coffee-Mountain-Sky-742

    Additional Security Measures:

    • Password Managers: LastPass, 1Password, Bitwarden
    • Two-Factor Authentication: Google Authenticator, Authy
    • Regular Updates: Change passwords every 3-6 months
    • Breach Monitoring: Use services like Have I Been Pwned

    Your passwords are never stored or transmitted. All checks are performed securely.
